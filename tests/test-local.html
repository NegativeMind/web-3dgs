🔧 ローカルdevテスト

このページは、npm run dev 実行中に動作確認するページです。

使用ファイル: /widget/index.ts（Vite が TypeScript を直接変換・ビルド不要）

URL: http://localhost:5173/tests/test-local.html

Object モード（OrbitControls）

確認項目：
  • <threedgs-viewer> Custom Element が登録・描画されるか
  • Shadow DOM によるスタイル分離が機能しているか
  • マウスドラッグで回転、ホイールでズームできるか
  • ロード中インジケーターが表示されるか
  • ResizeObserver によるキャンバスリサイズが機能するか
💡 ヒント： F12 → Console でエラーや初期化ログを確認できます。

Immersive モード（FPS 移動）

確認項目：
  • クリックでポインターロックが有効になるか
  • WASD / 矢印キーで移動できるか
  • マウス移動で視点が変わるか
  • WebXR 対応デバイスで VR ボタンが表示されるか

確認項目：
  • 異なるサイズで正しく描画されるか
  • 複数インスタンスが独立して動作するか
