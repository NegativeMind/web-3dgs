🧪 CDN配信テスト

このページは、jsDelivr から配信されるウィジェットの動作確認用です。

使用バージョン: beta v0.0.0（タグ: @v0.0.0-beta）

CDN URL: https://cdn.jsdelivr.net/gh/NegativeMind/web-3dgs@v0.0.0-beta/dist/3dgs-viewer.js

⚠️ タグ v0.0.0-beta を GitHub で作成してからでないと CDN から配信されません。

Object モード（OrbitControls）

確認項目：
  • <splat-viewer> Custom Element が登録・描画されるか
  • Shadow DOM によるスタイル分離が機能しているか
  • マウスドラッグで回転、ホイールでズームできるか
  • ロード中インジケーターが表示されるか
  • ResizeObserver によるキャンバスリサイズが機能するか
📋 埋め込みコード例：
<splat-viewer src="https://negativemind.com/3dgs/gmk.sog" scene-type="object" width="800px" height="480px"> </splat-viewer> <script src="https://cdn.jsdelivr.net/gh/NegativeMind/web-3dgs@v0.0.0-beta/dist/3dgs-viewer.js"></script>
💡 ヒント： F12 → Console でエラーや初期化ログを確認できます。CDN からのファイル取得は Network タブで確認できます。

Immersive モード（FPS 移動）

確認項目：
  • クリックでポインターロックが有効になるか
  • WASD / 矢印キーで移動できるか
  • マウス移動で視点が変わるか
  • WebXR 対応デバイスで VR ボタンが表示されるか

サイズ変動テスト

確認項目：
  • 異なるサイズで正しく描画されるか
  • 複数インスタンスが独立して動作するか
