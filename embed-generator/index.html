3D Gaussian Splatting Viewer

任意のWebサイトに3D Gaussian Splattingコンテンツを埋め込み可能なウィジェット

GitHub
使い方 3DGS ファイル（.sog / .ply / .splat）の URL を入力して「埋め込みコードを作成」を押してください。生成されたコードを任意の Web ページに貼り付けるだけでウィジェットを埋め込めます。
